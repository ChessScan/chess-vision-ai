### Generator: `generate_chess_v2.py`

**Features:**
- ✅ **Camera angles: 5-25°** - genuinely shallow phone perspective
- ✅ **Camera distance: 15-35cm** - physical phone positioning
- ✅ **Real chess positions** - FEN-based starting positions, middlegames, endgames
- ✅ **Pieces rendered** - all 32 pieces via Unicode glyphs with proper colors
- ✅ **Wood variations** - walnut, cherry, maple, ebony boards
- ✅ **Lighting modes** - natural, warm, cool, studio, evening
- ✅ **Metadata exported** - FEN + camera params per image for training labels

**Sample positions included:**
- Starting position (1.e4, Sicilian)
- Italian Game (7 moves deep)
- Complex middlegames (move 11)
- Endgames (King vs King, pawn races)

### Location
- **Dataset:** `/shared/datasets/synthetic_renders_v2/`
- **Generator:** `/generate_chess_v2.py`
- **Format:** 800x600 PNG with per-pixel metadata

### Location
**Container:** `chess-vision-ai/docker/blender-synth/`
- **Dockerfile:** CUDA 12.1 + Blender 4.0
- **Script:** `scripts/ultra_shallow_render.py`
- **Assets:** `data_generation/assets/pieces/` (3D .obj models)

### Technical Specs
**Camera (Ultra-Shallow Phone Perspective):**
- **Angle: 5-25° from horizontal** (phone at table edge)
- **Distance: 15-35cm** from board
- **Focal length: 26mm** (iPhone equivalent)
- **Hand jitter: ±3° tilt/roll** (realism)

**Renderer:**
- Blender Cycles (ray-traced shadows, reflections)
- HDRI environment lighting
- Procedural wood materials (walnut, maple)
- 3D piece geometry (obj imports or fallback procedurals)
- 640x640 output with ground truth annotations

### Build Command
```bash
docker build -t chess-vision:blender docker/blender-synth/
docker run --gpus all -v ./output:/workspace/output chess-vision:blender -- --count 1000 --samples 256
```

### Output
- Images: `/output/images/chess_####.png`
- Metadata: `/output/ground_truth.json` with FEN + bounding boxes
